# Support — FrootAI

> Have a question? Stuck on something? Here's how to get help.

## 🆘 Quick Triage — Pick the Right Channel

| What you need | Where to go |
|---------------|-------------|
| **Bug report** (something is broken) | [GitHub Issues](https://github.com/frootai/frootai/issues/new?template=bug.yml) |
| **Feature request** (you wish FrootAI did X) | [GitHub Discussions → Ideas](https://github.com/frootai/frootai/discussions/categories/ideas) |
| **Question** ("how do I...?") | [GitHub Discussions → Q&A](https://github.com/frootai/frootai/discussions/categories/q-a) |
| **Talk to people** (chat, banter) | [Discord](https://discord.gg/frootai) (planned) |
| **Real-time docs lookup** | [Agent FAI chatbot](https://frootai.dev/chatbot) |
| **Security vulnerability** | [Security Policy](./SECURITY.md) — **never** open a public issue |
| **Code of Conduct concern** | conduct@frootai.dev |
| **Enterprise / commercial** | hello@frootai.dev |

## 🐛 Filing a Great Bug Report

A great bug report tells us:

```markdown
**Channel + version**:
e.g. `frootai-mcp@5.2.0` (npm) on macOS 14, Node 22.

**What you expected to happen**:
e.g. `npm run factory:website` should write 7 JSON files to `public/data/`.

**What actually happened**:
e.g. It writes 6 files; `cookbook.json` is missing.

**Reproduction steps**:
1. ...
2. ...
3. ...

**Logs / output**:
```
paste relevant output here, redact secrets
```

**Environment**:
- OS:
- Node / Python version:
- Other relevant info:
```

The faster we can reproduce, the faster we can fix.

## ⏱️ Response Time Expectations

We are a small open-source team. Please be patient and be kind.

| Channel | Typical First Response |
|---------|-----------------------|
| Security report | ≤3 business days (see [SECURITY.md](./SECURITY.md)) |
| Critical bug (broken release, data loss) | ≤2 business days |
| Regular bug | Within a week |
| Feature request | Reviewed in monthly triage |
| Q&A discussion | Often same-day from community |
| Agent FAI chatbot | Instant (24×7) |

If you have not heard back after the typical window, **do not open a duplicate
issue** — instead, leave a polite comment on the existing thread bumping it.

## 💬 Communication Etiquette

- Be specific. "It doesn't work" is not actionable.
- Be patient. Maintainers volunteer their time.
- Be kind. Read [`CODE_OF_CONDUCT.md`](./CODE_OF_CONDUCT.md).
- Be helpful. If you find an answer, share it back.
