FrootAI's unified search finds anything across the entire knowledge engine — instantly, offline.

### How to Search

1. **Ctrl+Shift+P** → "FrootAI: Search All"
2. Or click **🔍** in the sidebar header
3. Type your query — results appear live with color-coded categories
4. Select a result to jump directly to it

> **Tip**: Multi-word queries work — try `"enterprise rag azure"` for precise results.
