## Browse 101 Solution Plays

FrootAI ships **101 pre-architected AI solutions** — from Enterprise RAG to Swarm Orchestration, Edge AI to Content Moderation.

### What's Inside Each Play

| Layer | What You Get |
|-------|-------------|
| **DevKit** | Copilot agents, instructions, skills, hooks, prompts |
| **TuneKit** | Model config, guardrails, evaluation thresholds |
| **SpecKit** | FAI manifest, architecture docs |
| **Infra** | Azure Bicep (AVM modules) |

### How to Browse

1. Open the **FrootAI** sidebar (🌳 icon in the Activity Bar)
2. Expand **Solution Plays** — grouped by FROOT layer
3. Click any play → rich detail panel opens
4. Use **🔍 Filter** to search by name, Azure service, or category

### Quick Actions from Play Detail

- **Init DevKit** — scaffold .github Agentic OS for the play
- **Cost Estimate** — toggle dev/prod Azure pricing
- **WAF Checklist** — see Well-Architected Framework alignment
- **Azure Services** — view all required Azure resources

> **Tip**: Recently opened plays appear at the top of the tree under ⏱️ Recently Used.
